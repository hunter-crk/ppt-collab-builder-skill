# 页面模板与布局规范

## PPT基础规范

### 尺寸与边距
- **画布尺寸**: 13.33 × 7.5 英寸（宽屏16:9）
- **边距**: 0.5 英寸
- **标题栏高度**: 0.8 英寸
- **内容区域**: 0.5-12.83 (水平), 1.0-6.8 (垂直)

### 字体规范
- **标题**: 思源黑体 Bold 24pt
- **正文**: 微软雅黑 Regular 14pt
- **小字**: 微软雅黑 Regular 10-12pt
- **数据**: Consolas / 等宽字体 12-14pt

| 边距 | 上下左右各 0.5 英寸 |
| 标题字体 | 思源黑体 Bold 24pt |
| 正文字体 | 微软雅黑 Regular 14pt |
| 主色（标题/强调） | 根据项目风格画像填写 |
| 辅助色 | 根据项目风格画像填写 |
| 辅助色（增长） | #00B42A（绿色） |
| 警告色（下降） | #F53F3F（红色） |
| 页面背景 | 根据项目风格画像填写 |
| 内容容器 | 可为无框留白、细边框卡片、分区色块或图片承载区 |
| 表头背景 | #F2F3F5（浅灰） |
| 结论呈现 | 根据页面任务选择标题下结论、侧栏洞察、底部总结、浮层标注等 |

## 每页结论呈现规范（最终大纲必需）

最终大纲中必须包含以下格式的结论呈现规范：

| 位置 | 样式 | 用途 |
|------|------|------|
| 页面标题 | 可为顶部标题栏、左侧竖向标题、无栏标题、章节式大标题 | 根据项目风格和页面任务选择 |
| 内容区域 | 可为无框布局、卡片、分区色块、全幅图表、图文混排 | 承载图表、表格、图片和说明 |
| 底部"核心洞察"栏 | 浅蓝背景（#E8F3FF）+ 主色标题 | 单维度结论页（如单店效能） |
| 右侧"洞察卡片" | 白色背景卡片 + 浅灰边框 | 多图表页（如生意概览） |
| 三列底部各自结论 | 每个模块下方卡片内 | 分模块页（如三维选品法） |

### 页面布局基础规范
- **标题形态**：根据风格画像选择，不强制全页统一深色标题栏
- **内容区域**：优先服务页面任务，可使用无框布局、卡片、色块、分栏或全幅图表
- **卡片使用**：只在需要分组、对比、承载指标时使用；不要所有内容都套卡片
- **间距**：模块之间保持0.16-0.3英寸间距，数据密集页可更紧凑，提案/品牌页应更舒展
- **统一性**：颜色、字体、数字样式、图表轴线风格保持统一；版式可以按页面任务变化

## 版式变体选择

页面类型不是固定模板。Agent必须先判断页面任务，再从下列变体中选择或组合；详细决策规则见 `adaptive-layout.md`。

### 通用版式变体

| 变体 | 结构 | 适用场景 |
|------|------|----------|
| `hero_kpi` | 大结论 + 1-3个核心数字 + 少量说明 | 首页、总结页、关键结论页 |
| `dashboard_grid` | 多指标/多图表网格 | 管理层经营看板、数据密集概览 |
| `chart_focus` | 单一大图表 + 简短洞察 | 趋势、结构、分布分析 |
| `split_insight` | 左主图/表 + 右洞察或行动建议 | 分析页、对比页、诊断页 |
| `before_after` | 左右对照或上下对照 | 改善前后、竞品对比、方案比较 |
| `matrix_quadrant` | 象限/矩阵 + 重点标注 | 增长机会、优先级判断、选品策略 |
| `image_catalog` | 图片网格 + 少量指标/标签 | 商品、品牌、案例、SKU展示 |
| `narrative_steps` | 横向或纵向路径图 | 策略、流程、阶段计划 |
| `dense_table` | 全幅或主区域表格 + 局部高亮 | 明细清单、多维指标、精确数值展示 |
| `section_divider` | 大标题 + 背景/图形/关键句 | 章节过渡，控制整份PPT节奏 |

### 版式节奏要求

- 6页以上的PPT，应至少使用3种不同版式变体
- 连续3页不要使用完全相同的标题形态和图表/洞察位置
- 数据密集页之后，优先安排结论页、分组页或留白更强的页面缓冲
- 同一章节内保持视觉一致，不同章节可以通过标题形态、背景色或布局节奏轻微变化
- 如果用户提供参考截图，优先提炼截图中的版式规律，而不是只复制颜色

## 页面类型详解

### 1. overview - 生意概览

**适用场景**: 首页、摘要页、关键指标总览

**可选版式**:
- `hero_kpi`：适合开场总结，用一个主结论带3-5个核心指标
- `dashboard_grid`：适合经营看板，用指标卡、迷你趋势图、占比图组合
- `split_insight`：适合客户汇报，左侧概览数据，右侧突出业务判断

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│  │ 指标卡片 │  │ 指标卡片 │  │ 指标卡片 │ │
│  └──────────┘  └──────────┘  └──────────┘ │
│                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│  │ 指标卡片 │  │ 指标卡片 │  │ 指标卡片 │ │
│  └──────────┘  └──────────┘  └──────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

**指标模块规范**:
- 尺寸: 3.8 × 2.5 英寸
- 间距: 0.5 英寸
- 模块内容:
  - 标签 (14pt, 浅灰)
  - 数值 (32pt, 粗体, 主题色)
  - 变化率 (14pt, 绿/红)

### 2. efficiency - 单店效能

**适用场景**: 趋势分析、时间序列数据、效能追踪

**可选版式**:
- `chart_focus`：单一趋势图占主视觉，适合讲清变化趋势
- `split_insight`：图表 + 右侧原因拆解或动作建议
- `dashboard_grid`：趋势图 + 关键分解指标，适合经营复盘

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌───────────────────────────────────────┐ │
│  │                                       │ │
│  │          大图表区域                   │ │
│  │        (折线/柱状图)                 │ │
│  │                                       │ │
│  └───────────────────────────────────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

**图表规范**:
- 图表尺寸: 11.8 × 5.2 英寸
- 支持类型: 折线图、柱状图、面积图
- 字体: 坐标轴14pt, 图例14pt, 标题20pt

---

### 3. structure - 品类结构

**适用场景**: 占比分析、品类构成、渠道分布

**可选版式**:
- `chart_focus`：饼图/环图/条形结构图占主区域，适合结构结论明确的页面
- `split_insight`：左结构图 + 右明细表/洞察
- `before_after`：结构变化前后对比

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌──────────────────┐  ┌─────────────────┐ │
│  │                  │  │                 │ │
│  │    饼图区域      │  │    数据表格     │ │
│  │   (占比展示)     │  │   (详细数据)    │ │
│  │                  │  │                 │ │
│  └──────────────────┘  └─────────────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

**双栏布局**:
- 左栏: 饼图 (5.5 × 5.2 英寸)
- 右栏: 数据表格 (5.9 × 5.2 英寸)
- 间距: 0.4 英寸

---

### 4. essential - 必备品全景

**适用场景**: 商品展示、SKU列表、必备品矩阵

**可选版式**:
- `image_catalog`：商品图片网格为主，适合展示SKU和案例
- `dense_table`：商品指标表为主，适合精确筛选
- `split_insight`：左商品矩阵 + 右侧选择标准/推荐理由

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ │
│  │商品 │ │商品 │ │商品 │ │商品 │ │商品 │ │
│  │图片 │ │图片 │ │图片 │ │图片 │ │图片 │ │
│  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘ │
│  名称   名称   名称   名称   名称    │
│                                             │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ │
│  │商品 │ │商品 │ │商品 │ │商品 │ │商品 │ │
│  │图片 │ │图片 │ │图片 │ │图片 │ │图片 │ │
│  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘ │
│  名称   名称   名称   名称   名称    │
│                                             │
└─────────────────────────────────────────────┘
```

**网格规范**:
- 列数: 3-5列 (根据商品数量)
- 卡片尺寸: 每列宽度均匀分配
- 图片高度: 1.2-1.5 英寸
- 名称标签: 7-10pt 小字

### 5. super_3d - 超单3D选品

**适用场景**: 数据表格、详细指标、多维度对比

**可选版式**:
- `dense_table`：明细表格为主，适合多指标精确比较
- `matrix_quadrant`：三维/四象限选品，适合突出优先级
- `split_insight`：表格 + 选品结论/风险提示

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌───────────────────────────────────────┐ │
│  │  表头  │  列1 │  列2 │  列3 │ ...  │ │
│  ├────────┼──────┼──────┼──────┼───────┤ │
│  │  行1   │  值  │  值  │  值  │  ...  │ │
│  │  行2   │  值  │  值  │  值  │  ...  │ │
│  │  ...   │  ... │  ... │  ... │  ...  │ │
│  └───────────────────────────────────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

**表格规范**:
- 表头: 12pt, 粗体, 灰色背景
- 内容: 11pt, 常规
- 斑马纹: 隔行浅灰背景
- 对齐: 数值右对齐，文本左对齐

---

### 6. super_performance - 超单品表现

**适用场景**: 商品对比、竞品分析、性能追踪

**可选版式**:
- `before_after`：商品或竞品对照
- `split_insight`：主对比图 + 右侧关键发现
- `chart_focus`：单一对比图强化表现差异

**基础布局结构**:
```
┌─────────────────────────────────────────────┐
│  标题栏                                     │
├─────────────────────────────────────────────┤
│                                             │
│  ┌──────────────────┐  ┌─────────────────┐ │
│  │                  │  │                 │ │
│  │   主图表区域     │  │   洞察卡片区    │ │
│  │   (对比分析)     │  │   (关键发现)    │ │
│  │                  │  │                 │ │
│  └──────────────────┘  └─────────────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

**左右布局**:
- 左栏: 主图表 (7.5 × 5.2 英寸)
- 右栏: 洞察卡片 (4.0 × 5.2 英寸)

---

### 7. targets - 目标设定

**适用场景**: KPI展示、进度追踪、目标完成度

**可选版式**:
- `hero_kpi`：突出目标完成率和差距
- `dashboard_grid`：多个KPI进度并列
- `narrative_steps`：目标拆解为阶段路径

---

### 8. growth - 增长机会

**适用场景**: 象限分析、散点图、机会识别

**可选版式**:
- `matrix_quadrant`：大象限图 + 重点机会标注
- `split_insight`：机会图 + 行动列表
- `narrative_steps`：从洞察到行动路径

---

### 9. investment - 投资回报

**适用场景**: ROI分析、投入产出、效益评估

**可选版式**:
- `before_after`：投入前后或方案对比
- `chart_focus`：ROI/瀑布图为主
- `split_insight`：财务图表 + 投资建议
